BUnwarpJ image registration with DEMs

Jason Goetz

2020-09-24

This vigenette is a short tutorial showing how to apply BUnwarpJ plugin image registration to digital elevation models (DEMs).

Install Fiji

Install Fiji

Open hillshade models

File > Open May have trouble running with GEOTIFF –have to convert to PNG. – e.g with IrfanView (save with no compression) (Remember order e.g. first 2012 than 2017) First, rock_glacier_hillshade_2012_50cm.tif Then, rock_glacier_hillshade_2017_50cm.tif

Open bunwarpJ plugin and map landmarks

Plugins > Registration > bUnwarpJ

Map landmarks on 2012 and adjust in 2017 (stable and active areas)

Save landmarks

Save landmarks bunwarpJ I/O Menu > Save Landmarks As…

Name the file to remember the direction of mapping (i.e. 2012 to 2017; Landmarks_source2012_to_target2017.txt).

Note: When loading landmarks – make sure that source image is 2012 and the target 2017, otherwise the landmarks will not be positioned correctly.

Run registration

Can use the following settings (Note: mapping/loading landmarks will automatically use them for registration).

Save transformation task will pop up when registration is complete.

Convert transformation to RAW

This transformation file can be opened in R using the bunwarpj(). Re-name to include RAW just for file management. In the I/O Menu select Convert Transformation to RAW 1. First select the transformation file 2. Write the filename – can be confusing because it says OPEN instead of convert/save
